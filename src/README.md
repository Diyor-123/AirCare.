# AirCare – забота о лёгких в условиях плохого воздуха

Простая платформа:
• Тест на состояние лёгких (0–30 баллов)
• Проверка качества воздуха (AQI)
• Рекомендации: упражнения или визит к врачу

## Текущий статус

- [ ] Подсчёт баллов и категории
- [ ] Получение AQI (Ташкент)
- [ ] Рекомендации + упражнения
- [ ] Чат-бот (DeepSeek / Gemini / GPT)
- [ ] Связь с сайтом на Lovable

Технологии: Python + JSON + AQI API
